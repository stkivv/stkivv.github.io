Quiz webapp (in development)

The aim of this project is to create a webapp that would let users create quizzes and play them in real time with others (sort of like Kahoot). For the backend I am using Spring Boot with a PostgreSQL database and for the frontend I am using Angular with Tailwind.

The project is currently in development. As of writing this, I have implemented authentication (for which I use JWT stored in httpOnly cookies) and everything needed to create quizzes.
